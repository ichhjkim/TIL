# 01_Java 프로그래밍이란?



## 1-1: 프로그래밍이란?

컴파일러: 소스를 기계어로 변환



## 1-2: Java언어의 탄생

- 제임스 고슬링이 만듬

- Sun 마이크로시스템즈에서 발표

- Oak언어에서 시작해서 Java언어로 발전
- 가전제품에 탑재할 수 있는 프로그램을 개발하기 위한 목적



## 1-3: Java 언어의 특징



- 현재 Java언어의 장점

  - 객체지향언어로 기능을 부품화할 수 있다

  - `JRE`를 이용해서 운영체제로부터 자유롭다 

  - 웹 및 모바일 프로그래밍이 쉽다

  - `GC(garbage collector)`를 통한 자동 메모리 관리를 지원한다

  - 실행속도가 많이 개선되어 빨라졌다.

    

- 초창기 시절에 Java언어의 단점

  - 기존 C/C++에 비해서 속도가 굉장히 느림
  - 리소스(메모리, CPU)를 많이 사용



## 1-4: JDK 설치(Java Development Kit)

JDK ⊃ JRE ⊃ API ⊃ JVM

개발자는 JDK가 있어야 개발할 수 있고 사용자는 JRE만 설치되어 있으면 된다
